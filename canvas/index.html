1

Enable Canvas in Google Sheets How to turn it on

Canvas is part of the Gemini-powered AI features in Google Workspace. It's rolling out gradually across accounts — if you don't see it yet, check back next week.

  1. Open any Google Sheet where you have edit access (not just view)
  2. Look for the Gemini / "Ask Gemini" icon in the top-right of the toolbar (sparkle icon)
  3. If you see "Create with Canvas" or "Canvas" in the dropdown, you have access
  4. If not, ask your Workspace admin to enable Gemini for Workspace for your account, or wait for the gradual rollout
★ Canvas needs Gemini for Workspace. Personal Gmail accounts on a free plan don't have it yet — it's currently enterprise / Workspace-paid accounts first.
✓ Once enabled, every Sheet in your account has Canvas available. No per-sheet setup needed.
2

Structure your dataset Canvas only works as well as your data

Canvas builds dashboards from whatever it reads in the sheet. A messy dataset produces a messy dashboard. Before prompting, make sure your data follows these rules.

  1. One header row only. No merged cells, no multi-row headers. Canvas needs column names it can reference.
  2. One row = one record. Each row should be a single observation (one day of data, one campaign, one creative, etc).
  3. Date columns in real date format. Format the column as Date — not text that looks like a date. Canvas uses this for time-series and aggregations.
  4. Metric columns in number format. Revenue, clicks, impressions, CPA — all as numbers, not text. Remove currency symbols from the cell (format them as currency instead).
  5. Dimension columns with consistent values. "Meta Ads" and "meta ads" get counted as different dimensions. Clean these up first.
  6. No blank rows in the middle. Canvas stops reading when it hits a blank row. Leave blank rows only after the last data row.
✓ Rule of thumb: if your dataset looks like a clean database table, Canvas will build a clean dashboard from it. If it looks like a report, Canvas will struggle.
3

Prompt Canvas Open the Gemini panel and start describing

With Canvas enabled and your dataset clean, you open the Gemini panel (Ask Gemini → Canvas) and describe the dashboard you want in plain language. The sheet builds the charts, filters, and summaries directly in a new tab.

  1. Click the Ask Gemini icon (top-right)
  2. Choose "Create with Canvas"
  3. Select the data range you want to analyze (the entire table)
  4. Write a prompt describing the dashboard (see 6 patterns below)
  5. Canvas generates the dashboard on a new tab. Edit the prompt to iterate.
★ The more specific your prompt, the better the output. "Build a dashboard" is weak. "Build a monthly dashboard with revenue, orders, CPA, ROAS as KPI cards, a day-by-week heatmap, and a 30-day revenue trend line" is great.

6 Canvas prompt patterns

Copy, paste into the Canvas prompt box, and adjust the metric names to match your data columns.

1 · Monthly KPI dashboard
"Build a dashboard for the last month with Revenue, Orders, CPA, and ROAS as large KPI cards at the top, each with a week-over-week delta percentage. Below the KPIs, add a 30-day revenue trend line chart and a day-by-week heatmap showing revenue intensity."
The classic marketing monthly. Works for agencies reporting to clients or e-com teams reporting internally.
2 · Channel breakdown
"Build a dashboard that compares performance across channels. Group data by the Channel column. Show a stacked bar chart of spend by channel over the last 30 days, a table of CPA and ROAS per channel, and a pie chart of revenue contribution. Add a filter for date range at the top."
Classic cross-channel view. Works for teams running Meta + Google + TikTok side-by-side.
3 · Campaign ranking
"Build a campaign leaderboard dashboard. Sort campaigns by ROAS descending. Show a table with campaign name, spend, revenue, ROAS, and a sparkline of 14-day revenue per campaign. Add conditional formatting: ROAS above 3x in green, below 1.5x in red. Include filters for Channel and Date range."
Directly answers "which campaigns should I kill and which should I double down on this week?"
4 · Week-over-week scorecard
"Build a week-over-week scorecard. Compare the last 7 days vs the previous 7 days. For each metric (Revenue, Orders, CPA, ROAS, CTR, CVR), show current, prior, delta absolute, and delta %. Highlight any metric that moved more than 10% in either direction. Add a sparkline for each metric showing the last 4 weeks."
Monday-morning snapshot. The fastest way to see what moved over the weekend.
5 · Funnel breakdown
"Build a funnel dashboard. Starting with Impressions, then Clicks, then Sessions, then Add to Cart, then Orders. Show absolute numbers and step-to-step conversion rates at each stage. Compare this month vs last month to see where the funnel is leaking or improving. Add a stacked area chart of daily funnel volume."
Finds the exact stage where conversion drops. Useful when "the numbers are off" but you don't know where.
6 · Creative performance heatmap
"Build a creative performance heatmap. Rows = creative names, columns = 4 weekly periods. Cell value = CTR, with heat color intensity. Below the heatmap, add a table with total spend, total revenue, CPA, and ROAS per creative. Sort creatives by ROAS descending."
One-view creative audit. Identifies which creatives are fatiguing and which are consistently performing.
